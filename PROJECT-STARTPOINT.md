# Portfolio Final Starting Point

This ZIP is the consolidated baseline after the reference-collage direction was approved.

## Active files

- `app/page.js` — homepage collage
- `app/globals.css` — single consolidated stylesheet
- `components/Nav.js` — fixed header/footer menu
- `data/projects.js` — project data
- `app/projects/page.js` — project archive
- `app/projects/[slug]/page.js` — case study template
- `components/ProjectRow.js` — archive row component

## Global controls

Edit the top of `app/globals.css` first:

```css
:root {
  --bg
  --pink
  --yellow
  --orange
  --white-card

  --type-hero
  --type-section
  --type-card
  --type-body
  --type-meta

  --menu-size
  --menu-x
  --menu-top
  --menu-bottom
}
```
